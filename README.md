# Prompt Playground

This repository contains experiments with Large Language Models (LLMs), chatbot development, and prompt engineering using GPT-2 and LLaMA models.

## Files Overview

### prompt_playground.ipynb

This notebook contains text generation experiments using the GPT-2 model from Hugging Face Transformers. It demonstrates how GPT-2 generates responses from user prompts and explores different text generation settings.

### promptPlaygrounLlama.ipynb

This notebook contains text generation experiments using the LLaMA model. It demonstrates prompt-based interactions and compares the behavior of LLaMA with GPT-2.

### prompt_engineering_playground.ipynb

This notebook focuses on Prompt Engineering techniques using the LLaMA model, including:

* Zero-Shot Prompting
* One-Shot Prompting
* Few-Shot Prompting

The notebook compares model responses under different prompting strategies and highlights their effectiveness.

### chatbot.py

A basic chatbot implementation using GPT-2.

### chatbot_Llama.py

A chatbot implementation using the LLaMA model.

### memory_chatbot.py

A GPT-2 based chatbot with conversation memory support.

### memory_chatbot_llma.py

A LLaMA based chatbot with conversation memory support.

### prompt_results.csv

Contains generated outputs and results from prompt experiments.

## Technologies Used

* Python
* Hugging Face Transformers
* GPT-2
* LLaMA
* Jupyter Notebook

## Learning Outcomes

Through this project, I explored:

* Text generation using pre-trained language models
* Chatbot development
* Conversation memory implementation
* Prompt engineering techniques
* Comparison of GPT-2 and LLaMA model responses

My Experience with LLaMA and GPT-2

As part of my exploration of Artificial Intelligence and Natural Language Processing, I worked with both the LLaMA and GPT-2 language models using the Hugging Face Transformers library.

I used Hugging Face to load pre-trained models and generate text through Python. While working with GPT-2, I learned how text generation models predict the next word in a sequence and create coherent responses from user prompts. I experimented with parameters such as temperature, maximum token length, and sampling methods to control the generated output.

I also explored LLaMA-based models and compared their performance with GPT-2. LLaMA models generally provided more advanced conversational abilities and better instruction-following capabilities, while GPT-2 helped me understand the fundamental concepts behind modern large language models.

Through these projects, I gained practical experience in model loading, prompt engineering, chatbot development, and text generation. I also learned how to use the Hugging Face Transformers framework to build AI-powered applications.

Overall, working with LLaMA and GPT-2 strengthened my understanding of Natural Language Processing and modern AI systems, providing valuable hands-on experience in the field of Artificial Intelligence.
